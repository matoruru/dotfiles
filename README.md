# dotfiles
my dotfiles repository, on gentoo linux
## shell
- bash (default)
- fish
## window manager
- gnome
- xmonad (default)
## terminal emulator
- GNOME Terminal
  - Custom font(Monaco for Powerline Regular 11)
  - Pallete                   -> Solarized
  - Text and Background Color -> Solarized dark
- rxvt-unicode (default)
  - Color scheme              -> Solarized
## main editor
- vim
  - Color scheme              -> Solarized dark
